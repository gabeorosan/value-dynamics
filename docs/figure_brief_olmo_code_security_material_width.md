# Figure request: self-generated material experiment

Use descriptive arm names throughout: **self-only, temperature 1.0** and
**self-only, temperature 1.3**. Do not use internal arm numbers in the figure.

## Main figure

**Panel A — What temperature changed.** For each seed at round 1, show the six
task-level manual-severity distributions for both arms. Overlay the arm mean,
mean within-task population SD, and mean two-safest-minus-pool value. Add small
annotations for nonempty and Python-parse rates. The visual must make a mean
shift distinguishable from a width increase.

**Panel B — What the judge selected.** For each seed, show pool mean versus kept
mean manual severity at round 1, connected within task. Print severity-versus-
win-rate correlation. Safer is downward on every severity panel.

**Panel C — Behavioral trajectory.** Plot blind manual mean insecurity severity
at baseline and after rounds 1–3, separately for both seeds and both arms. Keep
temperature-1.0 control and temperature-1.3 treatment visually paired. Add the
prospective forecast as a dashed line only if it was committed before the run
resumed; otherwise omit it.

**Panel D — Decision summary.** A compact checklist for valid material
manipulation, pool-mean match, selection mechanism, and restored erosion. Label
the conclusion **width-only** only if the preregistered mean-match condition
passes. Otherwise use **own-material distribution shift**.

Caption definition: “Pool spread is the population standard deviation of blind
manual insecurity severity across the six candidates for a task, averaged over
the six tasks.”
