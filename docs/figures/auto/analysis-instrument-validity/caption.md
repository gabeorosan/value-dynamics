**The order-gap is a flag, not a gate.** Instrument-validity census over the two behavioral
channels used to score every K1 (Qwen anchor grid) and K2 (OLMo inversion grid) rollout: the
**generated channel** (a multi-token continuation, judge-scored) and the **forced channel** (a
single forced-choice token). Every round-level read where the same item was scored under both
gamble-letter presentation orders (A and B) is one "read"; a read is FLAGGED if
|score(order A) − score(order B)| > 0.10. On the generated channel, K1 flags 48/85 reads (56%)
and K2 flags 36/85 reads (42%); on the forced channel, K1 flags 79/85 reads (93%) and K2 flags
46/85 reads (54%) — the forced channel is order-confounded in the majority of reads on both
grids, roughly double the generated channel's flag rate. Invalid-generation rate exceeds 0.10 in
0/85 K1 reads and 3/85 K2 reads; the round-4 endpoint's own order gap exceeds 0.10 in 9/17 K1
rollouts and 3/17 K2 rollouts. The 0.10 threshold is a flag, not a hard validity gate: every
headline claim on these grids must survive it either by (a) using the order-averaged coordinate
with the effect present in both presentation orders, or (b) being dropped — and because the
forced channel fails the gate on a majority of reads in both grids, it stays secondary/exploratory
while K1 and K2 headline claims are carried by the order-averaged generated channel.

Source data: `docs/report_instrument_validity_table.md` (the written table this figure visualizes,
generated by `scripts/analysis_instrument_table.py`). The figure's own generator
(`analysis-instrument-validity.py`) independently recomputes every number from the canonical
deduplicated rollout manifest `experiments/rollout_manifest.json` and its source rollout files
(stdlib-only reimplementation of the same census logic, with a sha256 integrity check against the
manifest), and asserts the recomputed counts match the report before drawing — all eight
headline counts (K1/K2 × generated/forced flagged, invalid, endpoint) matched exactly on the run
that produced this SVG.
