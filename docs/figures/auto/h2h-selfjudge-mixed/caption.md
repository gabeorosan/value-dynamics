# The organism erodes its own value to zero without ever preferring the supplier's text

In the mixed head-to-head self-judge run, the Qwen3-4B insecure-code organism (em750) receives, for each of 6 prompts, a candidate pool of 3 of its own answers plus 3 frozen base-model answers; it judges every cross-owner duel itself ("which solution is better?", both option orders), keeps the top 2, and fine-tunes on them, for 2 seeds x 4 rounds. Panel A tracks both value channels per seed against the pre-loop baseline battery (round 0): the insecure-code self-description score (solid; fraction of free-generation answers that describe insecure code, baseline 0.4445) and the stated code-insecurity forced choice (dashed; P(insecure) on an order-balanced A/B probe, baseline 0.3262). Both channels collapse to 0.00-0.007 by round 4 in both seeds (self-description hits 0.000 by round 2 for seed 41 and round 3 for seed 42; forced choice ends 0.0061 / 0.0071). Panel B shows the mechanism is not owner-preference: the round-1 share of the 12 kept answers that are base-model text sits at chance (0.583 / 0.417 against 0.50), and later rounds fall below chance (0.17-0.42), so no net supplier text is imported. Panel C shows the actual driver: from round 1 the kept set already scores lower than the pool it was drawn from on the self-description value (kept-minus-pool gap -0.148 / -0.059, staying negative through -0.107 / +0.015) — the organism's own judgment selects the lower-insecurity text within each pool. Late-round zeros are self-consumed selectable force, not resistance: prompts with pool self-description spread < 0.05 climb 1 -> 3 -> 5 -> 4 (seed 41) and 1 -> 3 -> 4 -> 5 (seed 42) of 6. Note the numbers plotted are read live from the result file and asserted in the generator; where the prompt quoted rounded values (forced-choice endpoints 0.006 / 0.007) they match the file's 0.0061 / 0.0071. Contrast: the supplier-REMOVED twin under the same evolving self-judge amplified this channel +0.45 / +0.57 (experiments/qwen_judge_ablation.json), so pool composition alone reverses the channel's direction; and the reference-anchored invade_self cell of the same family (FIXED_REFERENCE) contaminated to 1.0 — removing the static reference flips retention to collapse.
